**Tabs** — underlined horizontal tab bar with a turquoise active indicator and optional count chips.

```jsx
<Tabs
  defaultValue="all"
  items={[
    { id: "all", label: "الكل", badge: 24 },
    { id: "official", label: "مخاطبات رسمية", badge: 8 },
    { id: "drafts", label: "المسودات" },
  ]}
  onChange={(id) => {}}
/>
```
